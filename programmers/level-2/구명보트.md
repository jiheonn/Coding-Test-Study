# 구명보트

```python
def solution(people, limit):
    answer = 0
    
    people.sort()
    
    total = len(people)
    count = 0
    
    left = 0
    right = total - 1
    while left < right:
        if people[left] + people[right] <= limit:
            count += 1
            left += 1
        right -= 1
    
    answer = total - count
    
    return answer
```
